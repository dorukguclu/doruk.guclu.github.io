College Board Notes

Collaboration enhances thinking with more people, incorporating diverse perspectives and races without bias.

Computing innovations can be both online or physical.

Essential questions to ask about computing innovations include:

Why do they exist?

What problems do they solve?

What new capabilities do they provide that were previously unavailable?

Types of inputs include tactile, audio, visual, and text, and many devices accept different types of inputs.

Outputs are triggered by inputs or code segments, functioning similarly to inputs but as results.

A program is a collection of statements, containing code segments.

The working style of programs can differ.

Programs are developed by investigators who identify:

Constraints

User concerns and interests

Program requirements

The development process involves:

Brainstorming

Storyboarding

Planning

Laying out the plan

Organizing and developing

A team of developers collaborates on programs, with each person having a specific role, and everyone deserving credit.

Code segments used in development must be properly documented:

Documentation occurs at:

The beginning is for listing specifications.

During the project, for tracking progress.

After the project, explain the overall process.

Some programming languages offer commenting, a form of documentation that doesn’t affect the code but helps future understanding.

Errors in programming are classified into 4 types:

Logic error: A mistake in the program or algorithm that causes the program to run incorrectly.

Syntax error: Similar to grammar errors, occur when the language rules aren’t followed.

Run-time error: This happens during the execution of the program, defined by each programming language.

Overflow error: Occurs when a computer tries to handle a number outside its defined range.

Syntax errors are typically easy to identify, as the language highlights the problematic line.

Logic errors are the hardest to find and often require test cases.

Border cases (the limits of the code) are also important to test.

Hand tracing helps with loops by writing out variable values to determine the correctness of outcomes.

Hand tracing is used for small code segments, while larger segments may need debugging programs.

Visualizations or debuggers may be used if programs don’t work as expected.

During the development process, testing involves:

Using defined inputs to verify that an algorithm or program produces expected outcomes.

Programs are tested with inputs at or beyond the extremes of input data.

Testing process:

Programmers test right after developing the program.

Once inputs and outputs are determined, they test, edit, and retest the program continuously.

After programmer testing, users test the program to refine it further.

Once refined, the program is finally released.

Swift Learning

Programming revolves around inputs and outputs, used daily in various contexts.

Data is crucial in programming:

It is a collection of different or the same values stored in a place.

With inputs of data, we can produce outputs of data.

Outputs are results or expressions derived from data inputs.

There are many programming languages, with Swift being one of the most important.

Naming objects minimizes confusion, which is known as abstraction or identifiers.

Identifying objects helps clarify their purpose, making debugging easier later on.

Camel case is the naming system used in Swift:

The first word remains lowercase.

The second word is capitalized, with no spaces between words.

In programming, the value of a text is called a string, made up of characters:

The order of the characters in a string is important.

Variables are used to store changing objects:

Variables must be declared first, indicating they are subject to change.

Constants refer to values that are not meant to change.

Values, constants, variables, and data can be used to create simulations:

Simulations recreate real-life scenarios.

By modeling with existing values and data, programmers can simulate variables and gather more data from these simulations.
